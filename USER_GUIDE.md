# SRT Tap Timer v1.55 ユーザーガイド

## 1. 概要

SRT Tap Timerは、音源を聴きながらキーボードで字幕タイミングを打ち、SRT字幕ファイルを作成・編集するローカルHTMLアプリです。

v1.55では、SRT作成に加えて、静止画・音源・字幕を使った簡易字幕付きWebM動画の作成機能を統合しています。横長動画とショート動画の両方に対応します。

推奨環境は Windows + Chrome / Edge です。

## 2. 起動

ZIPを展開し、`index.html` をChromeまたはEdgeで開きます。

GitHub Pagesで公開する場合も、基本的には `index.html` がアプリ本体です。

## 3. 歌詞・SRTの読み込み

「メタタグ付き歌詞テキスト / SRT下書き」から、歌詞テキストまたはSRTファイルを読み込みます。

歌詞テキストを読み込んだ場合は、「歌詞行を抽出」でメタタグ等を除いた歌詞行を抽出できます。抽出結果は「抽出・編集用テキスト」に入ります。

SRTを読み込んだ場合は、既存の開始・終了時刻と字幕テキストを字幕行リストに展開します。

文字化けする場合は、読込文字コードを変更して再読み込みしてください。通常は「自動」で問題ありません。

## 4. 音源の読み込み

「音源ファイル」から音源を読み込みます。WAVを推奨しますが、ブラウザが対応していればMP3も再生できます。

Sunoの無料ユーザーなど、MP3しか入手できない場合は、同梱の `convert_mp3_to_wav.bat` を使うとWAVに変換できます。ffmpegがPATHに通っている必要があります。

## 5. 基本操作

主なショートカットは次のとおりです。

- `Space`: 再生 / 停止
- `R`: 現在行の開始時刻をセットし、次の行へ進む
- `E`: 終了時刻をセットする
- `Z`: 元に戻す

歌詞字幕では、次の字幕開始まで前の字幕を表示し続ける方が自然な場面が多いため、R操作を中心にタイミングを作れます。明示的に字幕を消したい箇所だけEで終了時刻を入れます。

## 6. 選択行の編集

左側の「選択行の編集」では、現在選択中の行の開始時刻、終了時刻、歌詞を直接編集できます。

時刻欄は空欄、または `--:--,---` で未設定にできます。複数行字幕は `
` で表します。

編集後は「選択行に反映」を押してください。「選択行を再読込」を押すと、現在行の内容を編集欄に読み直します。

## 7. 字幕行リスト

右側の字幕行リストで、各行の開始時刻・終了時刻・歌詞を確認できます。

`＊` 列は、音源の現在位置で表示中の字幕を示します。

上級者モードでは、行追加、行削除、時刻消去、現在行以降の選択、選択行の一括シフトなどが使えます。

「字幕位置マーク更新周期(ms)」は、主にこの `＊` 表示や現在位置表示の更新周期です。動画作成の録画fpsや字幕タイミングには影響しません。

## 8. オフセットと波形

「タップ入力オフセット」は、R/Eで入力される時刻を前後に補正します。

「波形表示オフセット」は、周辺波形の表示位置を前後に補正します。

「波形表示を有効にする」をONにすると、現在位置の周辺波形を表示します。音源ファイルによっては解析に少し時間がかかります。

## 9. SRT保存

「SRT出力ファイル名」を確認し、「SRT保存」を押すとSRTファイルを保存できます。

出力名は基本的に、歌詞/SRTファイル名が優先されます。歌詞/SRTファイルがない場合は音源ファイル名が使われます。

## 10. プレビュー / 動画作成

右ペイン上部の「プレビュー / 動画作成」を押すと、動画作成ウィンドウが開きます。

動画作成では、本体側で読み込んでいる音源と、開始時刻・終了時刻の両方が入った字幕行を使います。動画作成ウィンドウで選択するのは画像ファイルだけです。

設定できる主な項目は次のとおりです。

- ショート動画: OFFなら横長 16:9 / 1280×720、ONなら縦長 9:16 / 720×1280
- 画像合わせ: 左端が全体表示、右に行くほど拡大
- 字幕の高さ
- 文字サイズ
- 文字に縁取り
- 背景に黒帯

動画プレビューでは、画像と字幕の位置を確認できます。音源・字幕プレビューは動画作成機能専用で、録画開始時には自動停止します。

## 11. WebM作成時の注意

「動画作成」を押すと、録画開始前に確認ダイアログが表示されます。

録画中にブラウザのタブを切り替えると、WebMファイル自体は作成されても、字幕が入らないことがあります。録画が終わるまで、必ずこのタブを前面に表示したままにしてください。

作成が完了すると「保存」リンクが表示されます。保存される動画はWebM形式です。

YouTubeにはWebMのままアップロードできる場合がありますが、TikTokやInstagramではMP4が必要になることがあります。その場合は、同梱の `convert_webm_to_mp4.bat` を使用してください。

## 12. 変換用BATファイル

### convert_webm_to_mp4.bat

WebMファイルをMP4に変換します。TikTok、Instagramなど、MP4が必要なサービス向けです。

WebMファイルをBATファイルにドラッグ＆ドロップしてください。同じフォルダに、同じベース名のMP4ファイルを作成します。

例: `sample_subtitled.webm` → `sample_subtitled.mp4`

### convert_mp3_to_wav.bat

MP3ファイルをWAVに変換します。Sunoの無料ユーザーがMP3しかダウンロードできない場合などに使えます。

MP3ファイルをBATファイルにドラッグ＆ドロップしてください。同じフォルダに、同じベース名のWAVファイルを作成します。

例: `song.mp3` → `song.wav`

### BATファイル共通の注意

どちらのBATも、ffmpegがPATHに通っていることを前提にしています。

また、同名の出力ファイルがすでに存在する場合は、上書きせずにエラーで停止します。

## 13. 既知の注意点

一度ファイルを読み込んだあと、もう一度「ファイルを選択」を開いてキャンセルすると、ブラウザのファイル入力欄が「選択されていません」と表示されることがあります。ただし、アプリ内部の読み込み状態は残っている場合があります。

これはブラウザ標準のファイル入力欄の挙動による表示上の違和感で、現時点では仕様として扱います。
