# SRT Tap Timer v1.53 experimental - User Guide

1. Open `index.html` in Chrome or Edge.
2. Load lyrics/SRT and audio as usual.
3. Use R/E timing, direct edit, undo, preview, and SRT export as in v1.51 experimental.
4. Open `Preview / Video Export β` to preview subtitles and create a simple WebM video.
5. When pressing `Create video`, confirm the warning dialog. Keep the browser tab in the foreground until recording finishes.
6. If you choose Cancel, recording does not start.

Video export uses the current audio and completed subtitle rows from SRT Tap Timer. In the video-export window, select only the still image and output settings.
